# esb
Automation ESB

### Automation enthusiast.

This repo is contains cross platform library automation based on robot framework which is built using python also appium for the bridge communication tools.

#### How to install for Windows:
1. Install Python
2. Install Robot Framework
3. Install Selenium Library (Web Automation)
4. Install Appium (Mobile App Automation)
5. Install VSCode / Android Studio (Install Extension Python,Robot Framework, Appium)
6. Install Git
7. Install Chrome Driver (Disesuaikan Chrome yang Eksisting Sekarang)

#### How to install for Mac:
1. Install Python
2. Install PIP
3. Check that the path for python and pip is added to the .bash_profile, if not add it manually
4. Install Robot Framework
5. Install Selenium Library (Web Automation)
6. Install Appium (Mobile App Automation)
7. Install VSCode / Android Studio (Install Extension Python,Robot Framework, Appium)
8. Install Git
9. Install Chrome Driver (Disesuaikan Chrome yang Eksisting Sekarang)




#### How to run automation :

##### Web

robot -d "test-results" web/Testcase/your-project-test
